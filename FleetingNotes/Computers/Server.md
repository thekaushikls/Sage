# Server
Created : 05-05-2023 09:51

1. A dedicated computer that provides services.
2. Multiple (client) computers can request (ask) the server for some service.
3. Usually is to
	1. send/receive information from web: #WebServer
	2. send/receive information from a storage: #DatabaseServer
	3. send/receive emails: #EmailServer
	4. send/receive files: #FileServer
4. Can handle one or more tasks simultaneously
5. Server is a #Role that a computer can take.
6. Servers need to be running 24x6, with little or no #Downtime because other computers / companies might be dependent on the service the server provides.
7. Servers have their own hardware (Server #Processors that support #Multiprocessing) and software (Server OS) setup.
8. Server needs to have multiple #HotSwap -able hard disks in a #RAID configuration, to prevent memory loss.
