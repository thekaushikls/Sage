# Top Down Design Approach
Created : 05-04-2023 08:57

Usually followed in 

1. Start with overall building, and set the big picture. This phase is almost similar to #LOD100 phase in the #BIM process.
	1. Site
	2. Context
	3. Visual / Architectural Style
	4. Overall Form
	5. Massing
	6. Client Preferences
2.  In the next phase, some details are worked out. This is the *Getting down to the details* step. This could lead to #LOD200 and so on.
3. These steps are repeated until most details are discussed, and agreed upon by various stakeholders.
