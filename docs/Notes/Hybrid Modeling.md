## About

The four main types of modeling are
1. [Boundary Representation Modeling](Boundary%20Representation%20Modeling.md)
2. Polygonal (Mesh) Modeling
3. [Constructive Solid Geometry Modeling](Constructive%20Solid%20Geometry%20Modeling.md)
4. [Voxel Modeling](Voxel%20Modeling.md)

Hybrid Modeling leverages the advantage of each of these four modeling techniques, and use them in appropriate areas.
