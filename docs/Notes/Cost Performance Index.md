# Cost Performance Index`
*Created* : 27-05-2023 09:38
<br>*Tags* : #CPI #Performance #Business
---

If the cost-to-performance ratio is high, it means that the cost is high and the performance is low. Usually not a desirable product.

Most companies try to keep the CPI as low as possible.
