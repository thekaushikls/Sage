# Document Verification (Schengen D)
Created : 06-04-2022 09:02

* Indian public documents (Birth Certificate, Marriage Certificate, etc.)are not valid / recognized in India by default.
* Public documents needs to be verified in Germany, to be used in Germany.
* Which documents need to be verified and which do not is decided at the discretion of the Consulate / Diplomatic Mission (at the country one is applying from).
* Document verification process will be initiated on behalf of German Authority, and conducted by authorized Lawyers. (They also carry their respective photo IDs)
* For Long Term VISAs, verification is usually required for specialty-cooks / job-seekers / family-reunion / marriage.
* Cost of verification depends on location of verification. It is advice to bring the amount of payment (if known prior to appointment). If verification is not required, the candidate will be informed.
