# Gaming PC
Created : 24-06-2022 08:33

* Gaming computers usually focus on high quality Video Card / GPU units, that make computing as well as performance better.
* Also tend to have high refresh rate (120Hz to 240Hz) which most online games demand.
* In a way, I believe that a gaming computer is a Workstation with a better GPU.
* The GPU helps handle Rendering, and most of the visualization tasks.
