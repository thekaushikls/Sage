## Conventions
1. Only full names will be used to name notes.
	   *Example: `Boundary-Representation-Modeling.md` will be used instead of `BREP.md`.*
2. <kbd> Hypens </kbd> are to be used while naming files[^2] to maintain consistency.
3. <kbd> References </kbd> are used for memoranda - references to other notes in this repository. They are usually inline like [this]().
4. <kbd> Footnotes </kbd> are used for opinions, and references from external sources. Check [Footnotes](index.md/#footnotes) section below for example.
