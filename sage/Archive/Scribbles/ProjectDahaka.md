
Project Dahaka is a time management console application, that helps to record timesheets.
Configuration, and Time-Sheet are stored in JSON, and CSV formats respectively.


Flow of Control
Program starts with background checks.
Background Checks:
	1. Does config file exist? If no, create it.
	2. Does CSV file exist? If no, create it.
